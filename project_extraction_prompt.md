# Portfolio Extraction Prompt

Use the prompt below with any AI tool (like ChatGPT, Claude, or myself) to cleanly extract information from raw project files (like project proposals, campaign reports, or raw copy) and format it perfectly for insertion into your portfolio.

***

### The Prompt

```text
You are an expert PR and Digital Marketing copywriter. I am building a professional portfolio and I have attached raw materials regarding one of my past projects.

Please review the attached materials and extract/synthesize the information into the exact JSON-like structure below. The content should be professional, concise, and focused on strategic impact and measurable results.

Remember my core focus areas: strategic communication, digital engagement, and PR campaigns.

Please format your response EXACTLY like this:

**Category**: [Short category like 'Startup Project', 'Sports Marketing', 'Graduation Project']
**Year**: [e.g., '2023-2024']
**Title**: [The official name of the campaign or project]
**Project Type Badge**: [Choose one: 'Video Campaign', 'Brand Identity', 'Influencer & PR', or 'Social Media']

**Impact Statement (1 paragraph)**:
[Write a strong 2-3 sentence paragraph starting with an action verb (e.g., "Transformed", "Enhanced"). Focus purely on the high-level business/strategic impact and any measurable audience growth or brand recognition achieved.]

**Description (1 paragraph)**:
[Write a 2-3 sentence paragraph detailing the exact tactical execution. What did I do? Include keywords like 'competitor analysis', 'visual style definition', 'content scheduling', 'sentiment patterns', etc.]

**Highlights / Skills (Choose ONE format below)**:

*Format A (For quantifiable outcomes):*
- Number 1: [e.g., "45+"] -> Label 1: [e.g., "Designs Created"]
- Number 2: [e.g., "15+"] -> Label 2: [e.g., "Media Outlets"]
- Number 3: [e.g., "8"] -> Label 3: [e.g., "Partnerships"]

*Format B (For skill pills):*
- Pill 1: [e.g., "Brand Strategy"]
- Pill 2: [e.g., "Competitor Analysis"]
- Pill 3: [e.g., "Social Media"]
- Pill 4: [e.g., "Content Planning"]

*Format C (For a phased approach):*
- Step 1 Name: [e.g., "Audit & Analysis"] -> Step 1 Text: [1 sentence description]
- Step 2 Name: [e.g., "Community Engagement"] -> Step 2 Text: [1 sentence description]
- Step 3 Name: [e.g., "PR Campaigns"] -> Step 3 Text: [1 sentence description]

**Call to Action Link (Optional)**:
URL: [Link to social media page, case study URL, or LinkTree]
Button Text: [e.g., "Social media page"]
```
